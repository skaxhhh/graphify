시작(start) → "실행 이력"에 새 run 생성. 중지(stop) → 해당 run 종료. 운영 결과 화면과 직접 연결.
확인 필요 — 가정 4가지 (planning 전 확정)
  1. "run" 정의: start→stop 1회 = 1 run, 재실행 = 새 회차. (맞나?)
  2. 실행 이력 리스트 컬럼 구성. (추가/제외?)
  3. 기간 필터: "이 실행만 ↔ 전략 전체 통합" 2-mode. (충분한가?)
  4. 전략(run)별 가상 서브계좌 vs 단일 계좌 공유. (대시보드 equity/cash 정의)
  5. 통합 메뉴 명칭: "운영 결과 > 실행 이력". (확정?)
