## What Was Built

**CandleSection.tsx** — self-contained section component reused by both pages:
- Props: `symbol`, `date`, `trades`, `indicators`, `highlightTime`
- `useQuery` on `['backtest','bars',symbol,date]` via `fetchBars`, `enabled: !!symbol && !!date`
- `useMemo` computes `indicatorLines` from `bars + indicators` (SMA/EMA per spec)
- 4 dark-themed inline states (intentional deviation from shared/):
  - No selection → "표시할 거래가 없습니다."
  - Loading → emerald-500 spinner
  - Error → red-500/10 banner + 재시도 button (`refetch()`)
  - Empty bars → "해당 일자의 5분봉 데이터가 없습니다."
  - Success → `<CandleChart>` wrapped in `data-testid="candle-chart"`
- `data-testid="candle-section"` on card, `data-testid="candle-chart"` on success wrapper

**PaperBacktestPage.tsx** updates:
- `selected` state `{ symbol, date, time }`
- `onSuccess` auto-selects first trade (locked §1); 0 trades → `setSelected(null)`
- `handleRowClick`: always `setSelected` + accordion toggle only when `hasRationale` (RESEARCH Pattern 7)
- Indicator derivation: `extractIndicators(ruleDef)` from the selected rule's definition
- `CandleSection` mounted AFTER 수익 곡선 card, BEFORE 고급 통계 (SC-4)

**PaperHistoryPage.tsx** updates:
- `selected` state + `useEffect` auto-selects first trade on `data` load
- `handleRowClick`: always `setSelected` + accordion toggle when `hasRationale`
- `candleTrades`: `PaperTradeHistoryItem[]` mapped to `BacktestTrade[]` shape (datetime=tradedAt)
- `CandleSection` mounted between `<h2>` heading and trade table; `indicators=[]`

**candle-chart.spec.ts** — Playwright e2e, all APIs mocked via `page.route()`:
- SC-2: `data-testid="candle-chart"` visible + `canvas` element present after backtest run
- SC-3: success state maintained; no error/empty text visible
- SC-4: equity curve heading y < candle-section y (boundingBox comparison)
- SC-5: clicking SELL row loads candles for day 2 (chart stays visible, no error)
- Result: 4/4 green in 25.8s
