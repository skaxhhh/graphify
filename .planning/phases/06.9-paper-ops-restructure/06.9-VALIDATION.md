## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | JUnit 5 + Spring Boot Test (기존 backend 테스트 인프라) |
| **Config file** | `backend/src/test/resources/application.properties` |
| **Quick run command** | `cd backend && ./gradlew test --tests "com.graphify.trading.paper.PaperRun*" --tests "com.graphify.trading.paper.PaperExecutorRunTag*"` |
| **Full suite command** | `cd backend && ./gradlew test` |
| **Estimated runtime** | ~60–120 seconds (full suite) |

프론트엔드 자동화 테스트 인프라 없음 — 기존 모든 Phase가 백엔드 단위 테스트만 사용. 프론트 동작(D2/D3/D4 IA·nav·탭·기간필터)은 Manual-Only(visual) 검증.

## Sampling Rate

- **After every task commit:** Run quick command (PaperRun* + PaperExecutorRunTag*)
- **After every plan wave:** Run full suite command (`./gradlew test`)
- **Before `/gsd:verify-work`:** Full suite must be green
- **Max feedback latency:** ~120 seconds
