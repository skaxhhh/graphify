# Phase 6.9 Plan 04: 프론트 IA 재구조화 (Nav 3그룹 + 위젯 + run API) Summary

- `paperApi.ts`: `fetchPaperRuns`, `fetchRunDashboard`, `fetchRunHistory`, `fetchRunReport` 4종 추가 (Wave 3 백엔드 계약 일치, history/report는 mode/from/to 쿼리스트링 옵셔널).

**Task 2 — 전략 운영 상단 시장·스케줄러 위젯 (commit da02115)**

- `TradingRulesPage.tsx`: `fetchPaperMonitor()` useQuery 추가. h1 아래·테이블 위에 `grid grid-cols-1 sm:grid-cols-2 gap-4` 2카드 — 시장 상태(TradeCard, open=trade-up dot / closed=trade-muted dot + "개장"/"폐장") + 스케줄러 최근 실행(TradeStatCard, schedulerLastRun 포맷 또는 "—"). 신호로그·오늘 거래 미노출 (D3). 에러/로딩 시 "—"로 graceful.
