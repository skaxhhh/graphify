## Sampling Rate

- **After every task commit:** `cd frontend && npm run build` (tsc 오류 0건 + vite 번들 성공) + 해당 task 화면의 grep 토큰 가드(아래 (c))
- **After every plan wave:** Full suite — `npm run build && npm run test:e2e` (candle-chart.spec.ts GREEN 필수) + 해당 wave 화면 시각 검증(browse + visual-verdict)
- **Before `/gsd:verify-work`:** Full suite green + 전 화면 4상태 시각 검증 통과 + cream 비회귀 확인
- **Max feedback latency:** ~40s (build 단독) / ~160s (build+e2e)
