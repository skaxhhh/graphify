## Primitive API Reference (for waves 02–05)

### TradeButton
```tsx
<TradeButton
  variant="primary" | "secondary" | "danger" | "ghost"  // default: primary
  size="sm" | "md"                                        // default: md
  disabled={boolean}
  loading={boolean}
  type="button" | "submit" | "reset"
  onClick={() => void}
  className=""
>
  children
</TradeButton>
```
- `primary` = yellow (#fcd535) + ink text, hover active, disabled desaturated
- `secondary` = bg-trade-surface + border-trade-hairline
- `danger` = bg-trade-down + white text
- `ghost` = transparent + hover bg-trade-elevated

### TradeBadge
```tsx
<TradeBadge variant="active" | "draft" | "running" | "stopped" | "up" | "down" | "info" | "warning">
  children
</TradeBadge>
```
- `active`/`running` = bg-trade-up-soft text-trade-up border
- `draft`/`stopped` = bg-trade-elevated text-trade-muted border-trade-hairline
- `up` = bg-trade-up-soft text-trade-up (no border)
- `down` = bg-trade-down-soft text-trade-down
- `info` = bg-trade-info/10 text-trade-info
- `warning` = bg-trade-primary/10 text-trade-primary

### TradeCard
```tsx
<TradeCard title="optional heading" className="">
  children
</TradeCard>
```
rounded-lg bg-trade-surface border-trade-hairline p-4

### TradeStatCard
```tsx
<TradeStatCard
  label="레이블"
  value="₩1,234,567"
  sub="부제 텍스트"
  valueColor="up" | "down" | "neutral"  // default: neutral
/>
```
value uses font-trade-mono; up=text-trade-up, down=text-trade-down, neutral=text-trade-on-dark

### TradeTable / TradeTableHeader / TradeTableRow
```tsx
<TradeTable className="grid grid-cols-5">
  <TradeTableHeader className="grid grid-cols-5">
    <span>col 1</span>...
  </TradeTableHeader>
  <TradeTableRow onClick={() => void} className="grid grid-cols-5">
    <span>cell</span>...
  </TradeTableRow>
</TradeTable>
```
Layout-agnostic: consumers pass grid-template via className. TradeTableRow has hover:bg-trade-elevated.

### TradePageState
```tsx
<TradePageState
  variant="loading" | "empty" | "error"
  title="제목"
  message="메시지"   // preserved as-is for e2e text assertions
  onRetry={() => void}
  className=""
/>
```
loading = animate-pulse skeleton block; empty/error = centered flex with optional retry link

### TradeInput
```tsx
<TradeInput
  mono={false}        // true → font-trade-mono for price/numeric fields
  placeholder="..."
  className=""
  {...standardInputProps}
/>
// forwardRef compatible
```
bg-trade-surface, border-trade-hairline, focus:ring-2 focus:ring-trade-info

### TradeModeIndicator
```tsx
<TradeModeIndicator />
// reads useTradingStore mode automatically — no props needed
```
Renders PAPER 모드 (dot=trade-primary) or 실거래 모드 (dot=trade-down) indicator card.
