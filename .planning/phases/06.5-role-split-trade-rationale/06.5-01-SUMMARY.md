## API Reference (Wave 2/3이 참조)

```
POST /api/v1/trading/paper/rules/{id}/activate   — DRAFT → ACTIVE (no backtest gate)
POST /api/v1/trading/paper/rules/{id}/deactivate — ACTIVE/STOPPED → DRAFT (RUNNING 시 ERR_LIFECYCLE_006)
POST /api/v1/trading/paper/rules/{id}/start      — ACTIVE/STOPPED → ACTIVE/RUNNING
POST /api/v1/trading/paper/rules/{id}/stop       — ACTIVE/RUNNING → ACTIVE/STOPPED (config ACTIVE 유지)
POST /api/v1/trading/paper/rules/{id}/copy       — 기존 유지
```
